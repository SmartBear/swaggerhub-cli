# Usage
<!-- usage -->
```sh-session
$ npm install -g smartbear/shub
$ shub COMMAND
running command...
$ shub (-v|--version|version)
shub/0.0.0 darwin-x64 node-v12.13.0
$ shub --help [COMMAND]
USAGE
  $ shub COMMAND
...
```
